\# Video Temporal Error Detector



\## Description

Detects frame drops and frame merges in video streams using:

\- Frame timestamp analysis

\- Motion consistency between frames



\## Features

\- Frame-level classification (Normal / Drop / Merge)

\- Annotated output video

\- CSV report



\## Requirements

\- Python 3.8+

\- OpenCV

\- NumPy



\## How to Run

```bash

pip install -r requirements.txt

python detector.py
